# CV Website

A responsive TypeScript CV site built with Vite.

## Run locally

1. Install dependencies with `npm install`
2. Start the site with `npm run dev`
3. Build for production with `npm run build`

## Customize

- Update profile content in `src/main.ts`
- Replace `public/cv.pdf` with your real resume PDF
- Add your final GitHub and LinkedIn URLs

## GitHub Pages

This repo is configured for GitHub Pages deployment through GitHub Actions.

After the first push:

1. Open the repository settings on GitHub
2. Go to `Pages`
3. Set the source to `GitHub Actions`

Your site will publish from the `main` branch workflow.
